Para crear un vector se debe de colocar al principo del archivo del código fuente la siguiente sentencia import

import java.util.*; Cuando creamos un vector u objeto de la clase Vector, podemos especificar su dimensión inicial, y cuanto crecerá si rebasamos dicha dimensión.

Para determinar el copnstructor se hace con la siguiente estructura, se colocla el tipo de parametro, la variable y luego se contatena con el new> y los parametros que se desea colocar en el ventor. Ejemplo:

int vector[]=new int[5]
